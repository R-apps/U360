# Setting up SMS Profile

### Creating a New Profile

To create a new profile,&#x20;

* Go to **Settings** > **SMS Settings**. Alternatively, press <kbd>Alt+S</kbd>&#x20;
* In the **SMS Settings** dialog, select **New** and give the new profile a unique name and select **Ok**. ![](<../../.gitbook/assets/Create SMS profile.png>)
* <br>
