# Exporting a Template

The **Export** feature allows you to generate usable outputs or move templates out of the designer for validation, sharing, or downstream use.

### Available Export Options

**Current Template**\
Exports the currently opened template as a JSON file.\
Use this when you want to back up the template or share it with another user or team.

**Server Templates**\
Exports templates that are already saved on the server.\
Use this to download approved or deployed templates for reference, reuse, or migration.

**Create PDF**\
Generates a PDF output from the template using the mapped data.\
Use this to preview the final customer-facing document, such as account statements, policy documents, or billing notices.

**Create HTML**\
Generates an HTML output of the template.\
Use this for interactive statements or digital communications like email statements, portals, or web-based delivery.

#### How to Export?

To export a template, go to **Template** > **Export**, and select the template to be exported in required format.&#x20;
