# Objects and Properties

All the objects are available in the left pane.&#x20;

To insert an object, simply click it. When you do, the platform opens up its basic ldisplay properties in the **Object** tab on the header menu, as you see here. After you place the object on the designer, use these properties to quickly adjust it's look and feel.&#x20;

For additional properties of any object, click the **Properties** tab in the right pane. This **Property Editor** displays the respective object properties. These properties vary for each object.&#x20;
